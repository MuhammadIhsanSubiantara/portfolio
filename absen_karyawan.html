Website Sistem Absensi Dan Penggajian Karyawan

Sistem absensi dan penggajian ini dapat mempermudah Admin dan karyawan dalam mengelola data absensi maupun data penggajian.Melalui sistem berbasis website ini, pengelolaan absensi dan penggajian ini dapat menghasilkan sebuah data absensi dan penggajian dengan sajian yang lebih baik sesuai kebutuhan.Admin dan Karyawan dapat mudah dalam mengakses data Absensi dan penggajian pada Botanica Laundy.Pengelolaan data absensi dan penggajian menggunakan platform website yang sangat membantu karena mudah untuk dioperasikan, selain itu dapat mempercepat dalam proses pencarian data absensi, penyortiran absensi per periode, penggajian karyawan , sehingga tidak membuang waktu dan dapat meningkatkan kinerja pada Botanica Laudry

Technology Used : PHP (CodeIgniter 3) , Bootstrap, MySQL
